# WAVPlayer

## 專案介紹

本專案為簡易 WAV 音效播放器，實作音效播放功能。

## 操作步驟:
1. 使用者點擊**瀏覽**按鈕載入 .wav 檔案
2. 點擊**撥放一次**，音樂將會撥放一次
3. 點擊**停止播放**，音樂會停止撥放
4. 點擊**重複播放**，音樂會重新開始重複撥放
5. 點擊**結束程式**

## 功能介紹:
1. 提供無滑鼠服務
2. 當音樂撥放時，會有動圖的產生 ; 當音樂停止時，會以靜態圖做表示
3. 有標籤顯示目前的播放狀態
